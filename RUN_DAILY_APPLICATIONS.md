# Run Daily Applications

Use this prompt in a Codex thread where the Chrome plugin is available and the Codex Chrome extension shows Connected.

```text
Use Chrome with my signed-in profile and follow this repository's job-application workflow.

Repository: Vineeshan94/Job_Apply

Read first:
- START_HERE_AUTOPILOT.md
- APPLICATION_PLAN.md
- RECENCY_PREFLIGHT.md
- TRACKER_SCHEMA.md
- AUTONOMOUS_RUNBOOK.md
- AUTOPILOT_BLOCKER_PLAYBOOK.md

Goal:
- Target up to 100 real-fit applications today when enough matching roles are available.
- Prioritize Java Full Stack, backend, AI, ML, Data Scientist, and Generative AI roles.
- Use H4 EAD as work authorization.
- If asked whether sponsorship is needed to work, answer No.
- Do not skip good roles only because they mention visa sponsorship, no sponsorship, or immigration support.

Use:
- Java/full-stack resume for Java, Spring Boot, backend, cloud, microservices, platform, and full-stack roles.
- Generative AI resume for AI, ML, Data Scientist, analytics, LLM, RAG, and data platform roles.

Submit when:
- The role is real, open, relevant, and not a duplicate.
- Required fields can be answered truthfully from the repo, resume, job description, LinkedIn/GitHub context, or safe judgment.
- No hard-stop blocker is present.

Hard stops:
- CAPTCHA, hCaptcha, reCAPTCHA, or anti-abuse pages.
- Passwords, credentials, security questions, or payment requests.
- SSN, DOB, passport, driver's license, tax, bank, or other sensitive identity fields.
- Legal attestations, background checks, drug screens, assessments, video/audio responses, references, or private facts requiring Vineesha.
- Suspicious, scam-like, or low-quality applications.

After each role:
- Check Gmail for confirmation when practical.
- Update JOB_APPLICATION_TRACKER.xlsx.
- Record submitted, already-applied, skipped, blocked, or Gmail draft status.
- Do not store passwords, tokens, Gmail codes, or private email content.

Summarize:
- Submitted count.
- Gmail confirmations found.
- Drafts created.
- Blockers needing Vineesha.
- Best next actions.
```
